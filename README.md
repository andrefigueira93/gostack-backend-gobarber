## Endpoints
### Appointments
- [x] **List appointments:** ``[GET] - /appointments``

- [x] **Create appointments:** ``[POST] - /appointments``
- [ ] **Update appointments:** ``[PUT] - /appointments/:id``

- [ ] **Delete appointments:** ``[DELETE] - /appointments/:id``
